Starting APPLICATION
--------------------------
1. Open the project folder and copy the folder path. 
	(For this its 'C:\Users\Administrator\Desktop\ALL-NODES\My-Weather-App')

2. Open Node cmd terminal and navigate to the project folder
	cd C:\Users\Administrator\Desktop\ALL-NODES\My-Weather-App

3. Run the Application
	npm start

4. Open browser and access project on localhost, port 3000.
	localhost:3000
